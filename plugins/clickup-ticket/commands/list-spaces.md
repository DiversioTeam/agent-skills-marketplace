description: Discover your ClickUp workspace structure - spaces, folders, lists, tags.
---

Use your `clickup-ticket` Skill in **list-spaces** mode.

Displays your workspace hierarchy with IDs for reference:
- Spaces → Folders → Lists
- Available statuses per list
- Tags per space

**Flags:**
- `--org=<slug>` - Show different org
- `--tags` - Include tags for each space
- `--members` - Include team members
- `--all` - Show everything

Useful for finding list IDs or verifying cache is up-to-date.

See the SKILL.md for full details.
