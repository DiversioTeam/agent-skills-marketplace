# Upstream Provenance

This skill adapts reusable reference material and HTML templates from the
upstream project:

- Repository: `https://github.com/nicobailon/visual-explainer`
- Upstream commit used for import:
  `f41c48fd507c95f53151047788662e91def8c673`
- Upstream repository license: MIT

Copied or adapted upstream assets in this skill:

- `references/css-patterns.md`
- `references/libraries.md`
- `references/responsive-nav.md`
- `references/slide-patterns.md`
- `templates/architecture.html`
- `templates/data-table.html`
- `templates/mermaid-flowchart.html`
- `templates/slide-deck.html`

Diversio-specific additions in this repo include:

- the repo-native `SKILL.md`
- the interactive `/visual-explainer:explain` command wrapper
- the stakeholder explainer mode guidance
- marketplace metadata and installation docs
